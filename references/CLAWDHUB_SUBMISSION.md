# ClawdHub Submission Draft

## Title
macos-calendar-assistant

## One-line description
OpenClaw-native macOS calendar skill for IM-first schedule management with safe idempotent writes and duplicate cleanup.

## Long description
`macos-calendar-assistant` is a practical calendar execution layer for OpenClaw.
It helps close the loop between weekly planning, daily execution, daily review, and schedule adjustment.

It is designed for IM-first usage (Telegram/Discord/Feishu/iMessage/Slack workflows), supports screenshot-to-schedule style inputs, and keeps calendar writes safe through idempotent upsert + explicit cleanup confirmation.
